Medical AI — Nephrology Chat Assistant

This project implements a multi-agent medical AI assistant with:

  • Receptionist Agent
  • Clinical Agent (RAG-driven)
  • Patient identity verification
  • Chroma vector database with embeddings
  • FastAPI backend
  • Streamlit frontend
